### The need for a "GPO Crawler"
Imagine a scenario where you are dealing with 50+ machines and you require the GPO Result of each machine every 2-3 hours. Having to manually crawl the GPo REsult of each machine is a hassle and requires an overwhelming time dedicated to it. By creating a script that can automate this whole process will not only be more efficient but it also allows it to be centralised.
